# 日語菜鳥村

給日語超級初學者的手寫描紅練習網頁（PWA）。目前包含**平假名 46 音＋片假名 46 音（共 92 字）**，首頁可切換平／片假名。之後會再加入數字、星期、月份、日期。

線上版：<https://jerrihuang.github.io/hiragana/>

## 特色

- **菜鳥養成**：一隻菜鳥小雞當你的分身，隨著學會的字數（花丸）從「迷糊蛋」一路變身到「村長」，共 10 段（見 `js/mascot.js`）。
- **描紅為主**：照著淡色筆畫，從 ① 開始描。系統會檢查筆順、方向與是否描到位，判定刻意放寬，對初學者友善。
- **日系文具手帳風**：原稿紙方格、朱色格線、手寫字體，寫完蓋「花丸（はなまる）」印章當獎勵。
- **給台灣初學者**：每個字附注音提示（あ → ㄚ）與一個初級單字。
- **發音**：使用瀏覽器內建語音（需系統有日文語音）。
- **小測驗**：學會 4 個字以上就能玩看字選音。
- **進度只存在本機**（localStorage），不需帳號、不需連網。

## 如何執行

因為用到 Service Worker 與模組載入，請用本機伺服器開啟（不要直接雙擊 index.html）：

```bash
python -m http.server 8123
```

然後用瀏覽器開啟 <http://localhost:8123>。

（在 Claude Code 中也可用內建預覽：`.claude/launch.json` 已設定好名為 `hiragana` 的伺服器。）

## 檔案結構

```
index.html              主頁面（單頁式，四個視圖切換）
css/style.css           手帳風樣式、深色模式
js/data.js              10 個假名的筆順路徑、注音、單字、五十音表
js/app.js               描紅引擎、筆順動畫、花丸、測驗、發音、進度儲存
manifest.webmanifest    PWA 設定
sw.js                   離線快取（network-first）
icons/icon.svg          App 圖示
```
